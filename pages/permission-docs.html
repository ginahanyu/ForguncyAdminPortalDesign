说明文档

AdminPortal 分级授权增强 - 按应用分组显示授权信息

功能概述

本功能为活字格管理控制台提供了按应用分组显示授权信息的能力，使管理员能够更清晰地查看和管理不同应用的权限分配情况。

主要特性

  • 按应用对授权信息进行分组展示
  • 支持管理员和管理组两种视图切换
  • 提供详细的权限信息查看
  • 直观的左右分栏布局设计
  • 支持快速切换查看不同管理组的权限

界面说明

顶部标签页

包含"管理员"和"管理组"两个标签页，可以快速切换不同的授权视图。右侧的"+"按钮用于添加新的管理组。

左侧列表区

显示超级管理员分类下的所有管理组，点击可以查看对应管理组的详细权限信息。当前选中的管理组会高亮显示。

右侧详情区

分为两个主要部分：

  • 基础信息：显示管理员名称、管理角色和所属管理组
  • 权限信息：详细列出组织权限、应用权限、第三方权限等各项权限详情

使用场景

  • 查看特定管理组拥有的所有应用权限
  • 对比不同管理组之间的权限差异
  • 审计和管理应用的访问控制策略
  • 快速了解超级管理员的权限范围

技术实现

  • 采用纯前端HTML + CSS + JavaScript实现
  • 响应式布局设计，适配不同屏幕尺寸
  • 模块化的页面结构，便于维护和扩展
  • 基于路由的单页应用架构
